# بوت تلقرام لإدارة العملاء

## خطوات التشغيل

### 1. إنشاء البوت
- افتح تلقرام وتحدث مع @BotFather
- أرسل /newbot
- اختر اسم للبوت
- انسخ التوكن وضعه في bot.py مكان YOUR_BOT_TOKEN_HERE

### 2. احصل على ID حسابك
- تحدث مع @userinfobot
- انسخ الـ ID وضعه في bot.py مكان 123456789

### 3. تثبيت المتطلبات
```
pip install -r requirements.txt
```

### 4. تشغيل البوت
```
python bot.py
```

### 5. للتشغيل الدائم على سيرفر
```
nohup python bot.py &
```

## الميزات
- قائمة خدمات للعملاء
- نظام تذاكر لتجنب اللخبطة
- إشعارات فورية للأدمن
- أزرار رد ورفض للأدمن
- تأكيد تلقائي للعميل برقم تذكرته

## تعديل الخدمات
في bot.py ابحث عن SERVICES وعدّل الأسماء والأوصاف والأسعار حسب خدماتك.
